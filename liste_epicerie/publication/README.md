# SmartGrocery

Description brève du projet:
```text
Gestionnaire d'épicerie faite par un étudiant québécois en informatique.
```

Description complète du projet:
```text
Bonjour, je suis un étudiant en technique informatique qui avait comme 
travail final de faire une application de gestion d'épicerie pour personne seul, 
par famille ou par colocataire. 
Je rend public l'application pour les personnes qui veulent l'utiliser. 
SmartGrocery permet aux utilisateurs de faire des listes d'épicerie à partir 
d'articles globaux que les utilisateurs ont ajouté soit manuellement ou 
par le code CUP/UPC (le code barre). Ce n'est pas tout 
les articles à code barre qui existe sur le site https://world.openfoodfacts.org/. 
Le principe est que chaque utilisateur ajoute les articles qu'ils auraient besoin 
dans la liste globale si cet article n'est pas présent. 
Vous pourriez ensuite le mettre dans votre liste pour l'acheter lors de votre épicerie. 
VOUS NE POUVEZ PAS ACHETER L'ÉPICERIE VIA L'APPLICATION! 
Il y a, aussi, un historique d'épicerie qui est géré pour 
savoir qu'elle article vous avez déjà pris.

Vidéo démonstrative: https://www.youtube.com/watch?v=CvIxjzk62bs
Merci à: https://world.openfoodfacts.org/


N'ésiter pas à m'envoyer des suggestions pour améliorer 
l'application à mon adresse courriel: lokolokololo41@gmail.com 
```
